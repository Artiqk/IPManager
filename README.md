# IPManager
You will need a Debian with Aptitude to make this program work
## Install the dependancies
```
sudo apt install figlet sqlite3 libsqlite3-dev
```
Then you'll need to prepare the database
```
touch ip.db
sqlite3 ip.db < create.sql
```
## Compile
```
./compile.sh
```
## Execute
```
./ip_manager
```
